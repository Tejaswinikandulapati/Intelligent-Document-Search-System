# 📘 Intelligent Document Search System

An AI-powered web application developed using Python, Flask, NLP, and REST API concepts to upload, process, and search documents efficiently. The system performs intelligent keyword-based document retrieval with highlighted search results using NLP preprocessing techniques and provides a modern, user-friendly interface.

# 📂 Project Modules

1️⃣ Document Upload Module  
Allows users to upload text documents into the system.

2️⃣ NLP Processing Module  
Processes uploaded document text using NLP preprocessing techniques.

3️⃣ Keyword Search Module  
Searches documents intelligently based on user-entered keywords.

4️⃣ Highlight Result Module  
Highlights matching keywords inside retrieved documents.

5️⃣ User Interface Module  
Provides a clean and interactive interface for better user experience.
